# CLAUDE.md

Before editing any file in this repo or in any sibling gist repo,
read both:

- [style_code.md](style_code.md) -- language-neutral coding
  principles (COI, LoB, SSOT, PATHS, ASCII-only, naming vocab)
  plus per-language deltas it links
- [style_gist.md](style_gist.md) -- gist file structure + README
  (`,project.md`) format

Lua repos: also read
https://github.com/aiez/luamine/blob/main/docs/style.md
(file skeleton, help-string SSOT, `l.boot`, `-->` sig comments).

All are short. Follow them. Help/doc string is the SSOT for flags
and defaults. ASCII only. Terse ("smoosh") style in code AND docs.
